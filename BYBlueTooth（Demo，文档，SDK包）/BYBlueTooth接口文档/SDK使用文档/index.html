简介

BYBlueTooth使用文档

开发环境

BYBlueTooth使用Objective-C进行开发，开发环境为Xcode8.3，使用iOS10.3进行真机调试。

导入SDK

SDK使用.frameWork方式进行打包。在使用SDK时，只需要将 BYBlueTooth.framework 拉进项目中即可使用。

拉进项目时注意勾选“Copy items if needed”。

导入完添加下面语句即可使用：

//OC
#import <BYBlueTooth/BYBlueTooth.h>
//Swift
import BYBlueTooth
